How this reads at a glance
1 · Pick an Artifact
Grouped by trait type & rarity, like a stacked resource rather than 1-per-slot gear. Greyed out if this agent never rolled that trait.
2 · Preview the push
The trait's range bar gains a brighter overlay fill showing current → projected, clamped at the trait's real max. Already-maxed traits show "Already maxed" and can't be targeted.
3 · Apply, consumed
One Artifact, one application. No bulk-apply — each is a deliberate choice, same weight as a Crafting Bench re-roll today.
